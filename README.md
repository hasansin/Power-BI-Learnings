### Bank Churn Data
#### Description -
The Bank Churn dataset contains customer details such as credit score, geography, gender, age, tenure, balance, number of products, credit card status, activity status, estimated salary, and whether the customer exited (churned). It is mainly used to analyze customer behavior and predict churn.
